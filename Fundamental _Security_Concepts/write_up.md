## Overview
The purpose of this lab is to practice creating and managing virtual hard disks to support redundancy and data protection, reinforcing the concept of availability within the CIA Triad. This exercise demonstrates how storage configuration contributes to system resilience and continuity.


## Objectives
* Demonstrate how redundancy improves data protection and system resilience
* Create and configure Virtual Hard Disks (VHDs) in Windows 11
* Apply storage management techniques that support availability in the CIA Triad
* Gain hands-on experience navigating disk management tools in the Windows 11 OS
